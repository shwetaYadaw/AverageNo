# Average of Numbers in C++

## Project Overview

This project is a simple C++ program to calculate the average of numbers entered by the user.

The program takes input for the total number of elements, stores them in an array, finds their sum, and then calculates the average.

It is useful for beginners who are learning arrays, loops, and basic arithmetic operations in C++.

---

## Features

- Takes input for `n` numbers
- Stores numbers in an array
- Calculates the sum of all elements
- Finds the average of the numbers
- Displays the result clearly

---

## Technologies Used

- C++
- Arrays
- Loops
- Basic Mathematics
